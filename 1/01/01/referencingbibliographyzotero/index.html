2021-12-06 WORKFLOW WITH ZOTERO AND LATEX

PREFERENCES

  • SETTING HTML OPENER : XDG MIME QUERY FOR HTML FILES AND XDG-MIME DEFAULT FOR HTML FILES

  • ZOTERO PREFERENCES

    • general: auto take snapshots, auto attach, auto retrieve, auto rename, rename linked files, open pdf with llpp
    • advanced: data directory change

  • DOWNLOAD BETTERBITEX.XPI

    • zotero preferences -> export -> default format = betterbibtex quick copy
    • zotero preferences -> betterbibtex, export/quick-copy -> latex citation and parencite
    • zotero preferences -> betterbibtex, automatic export -> set onchange and set export unicode plaintext

  • DOWNLOAD ZUTILO.XPI

    • set alt q = focus item pane
    • set alt w = edit info
    • set alt c = show hide collections
    • set alt a = focus collections
    • set alt s = quick copy for citation key

WORKFLOW

  1. setup collection and export collection and add bibresource to tex file
  2. setup zoteroconnector and set extension shortcut and then save resources
  3. adjust items as necessary
  4. c-z to search and insert references, \parencite{} in tex file to list and insert

  • SETTING UP COLLECTION AND AUTOMATIC UPDATING

    • create new collection
    • click on collection
    • export collection -> betterbibtex automatic update
    • add bib file to tex header

  • COLLECTING RESOURCES

    • install zotero connector (always adds into latest collection)
    • setup chrome extension shortcut as (alt-z)

  • ENSURING RESOURCES ARE ACCURATE

    • alt q to focus item pane
    • arrow keys to navigate items
    • alt w to edit info
    • will auto export

  • VIM AUTO CITATION COMPLETE

    • \parencite | if texfile contains bibresource
    • with zoterocite and vim, opens up a zotero search and then you press enter twice and it gets inserted
    "copy into vimrc
function! ZoteroCite()
  " pick a format based on the filetype (customize at will)
  "let format = &filetype =~ '.*tex' ? 'latex' : 'pandoc'
  "let api_call = 'http://127.0.0.1:23119/better-bibtex/cayw?format='.format.'&brackets=1'
  let format = &filetype =~ '.*tex' ? 'latex' : 'latex'
  let api_call = 'http://127.0.0.1:23119/better-bibtex/cayw?format='.format.'&brackets=1.&command=parencite'
  let ref = system('curl -s '.shellescape(api_call))
  return ref
endfunction

noremap <leader>z "=ZoteroCite()<CR>p
noremap <c-z> "=ZoteroCite()<CR>p
inoremap <C-z> <C-r>=ZoteroCite()<CR>
