Recently I bought a second hand samsung galaxy tab 2 as a way to get some coding and reading done in more places.

It’s size is only 2.5 inches smaller than my smallest laptop, which is also my main workhorse (Thinkpad x260).

I did not like the original ROM running on it, so I decided to see if more performance could be squeezed out of it.

Installed Omnirom on it because, the omnilauncher as well as its shortcuts and tools felt amazing on my Redmi Note 2.

That did not work well. Afer a while

  1. Download odin

  2. Download twrp

  3. connect tab to pc

  4. boot into recovery mode

  5. flash twrp

  6. hold volume up and power while booting to get to twrp

  7. copy roms to base of storage

  8. reboot into twrp

  9. wipe

  10. install
