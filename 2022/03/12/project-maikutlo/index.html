Maikutlo TV series Tracking web site

Posted by Reece Petersen on Saturday, March 12, 2022

User manual – Maikutlo

Introduction

This user manual contains all the essential information that a user needs to fully utilise this information system. It includes descriptions of all functions, alternate use modes, procedures and processes to be followed in order to use and access all system functionalities. Step by step visual aids are also utilised for clarity.

Getting started

Maikutlo is a web-based series tracker. All you need is a device with a web browser that has a stable internet connection.

Landing page

When a user visits the Maikutlo website, this is the first page that welcomes them to the destination. It is a homepage of sorts.

Registration

On this page you get to create a user profile by filling out the registration form. The username is any name you would like to have displaying on your profile. The email and password supplied must have matching confirmation duplicates.

Email Verification

After pressing the register button an email with an account verification link will be sent to your email address. Do check your spam folder if the email does not reflect on your inbox. You can then copy and paste the link into your URL or click on the hyperlink if active.

Forgot Password

If you forgot your password don’t be worried. The forgot password button will direct you to the forgot password page so that a new one can be created again.

After requesting for a password rest, you will be directed to the login page.

Then check your emails for a reset password link. Do check your spam folder if the email does not reflect on your inbox. You can then copy and paste the link into your URL or click on the hyperlink if active.

Then, you will be directed on to the forgot password page for a new password. The new password must match with the duplicate password for verification purposes.

After clicking on the reset password button, the login page will be opened again so that you can login with the updated details.

Login

As a user with an existing account, the login button is selected which opens up the login page.

Trending page

The trending page will provide the trending series overall, and will update the results daily

Series search by series name

The search by series function is done by giving the users a section to input a series name, and the system would then search for related series according to that query and display them as cards. On each card various information can be found those being; series rating, series music, series trailer, series information.

Link to YouTube Trailers

Clicking on the YouTube icon on a series card will link you to the search for that particular series trailer.

Link to Whatsong music List

Clicking on the Music icon on a series card will link you to the corresponding page on Whatsong if it exists.

Series Details

Clicking on the Info (i) icon on a series card or clicking on the image of the series will take you to the corresponding series detail page.

This will contain a description of the series and synopsis of each season and episode if available.

Link to episodes music

Clicking on the YouTube and Music Button will link you to a YouTube search with the series title, season number and episode number.

Users Watch List

On any of the browsing pages be it trending, genre or search there will be a button to Add with a plus above it, it will only show when a user is logged in and does not already have the series in their watch list. Clicking on this button will add the series to the user’s watchlist.

The watchlist is accessible in the navbar.

Users Series Editing and Tracking

Once a user clicks on the Green edit button on the series card in the watchlist they will be redirected to the tracking and editing view where they can see details about their series and toggle episodes viewing status

Users Watch Time Statistics

After clicking the green link which says “Check your statistics” the user will be redirected to a statistics page which will show them their hours watched per series

Goal Setting

Once a user is successfully logged in, they would be taken to the profile page, which welcomes them and provides a section for setting goals according to anything they aim to do more series related to, for example they would want to watch a certain genre more

Edit Profile

While the user is successfully logged in the edit profile page is also visible where the user can edit their personal credentials, such as username, email and password; or delete an account if it no longer necessary

Top navigation button

As you scroll up and down the website a red button on the bottom left pops out written with the words “top”. This button makes it easy for you to jump straight to the top of the web page when clicked.

Glossary

Card: Refers to the rectangle within which series details and buttons are placed on the browse and watchlist pages

Watchlist: refers to the user view where series are stored
