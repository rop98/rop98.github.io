My Idea for SP 200

Requirements for project

Published Wed, May 20, 2020 by Reece Petersen

Estimated reading time: 1 min

Keeping track of series

  • User chooses genre or User searches via name
  • User chooses the series they follow to add to their follow list
  • a follow list of series is populated for user from selection
  • system counts the number of seasons and episodes
  • user marks how far along they are within series
  • system keeps track of how far along the user is
  • system notifies user when new episodes are released via email
  • User will be able to track educational series
  • System produces statistics based on watch time in the form of pie chart
  • Along with the series details there will be a link/list to/of the series music
  • If DOB under 18 then some genres will be restricted

languages: python, flask front-end: html, css scraping: beautifulsoup and/or selenium testing: pytest databases: mysql or sqlite

version control: git github gitkraken

resource_repo: google-drive diagramming: draw.io documentation: g-suite
