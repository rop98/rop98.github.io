  • I had an issue with images, specifically… I didn’t know where they go. I managed to read up that it is stored in the static folder. didn’t realise that static is considered the root. Eventually changed to folder name inside static and then image name and it worked

    • ![alt text](folder within static and then image name)

  • I then had an issue with figuring out how to determine the time it will take to read an article. This turned out to be incredibly simple, if the theme you’re using supports it.

Go to your config.toml or config.yml then in the params section add

[params]    
    **existing options**
    readingTime = true
    readingTimeText = "Estimated reading time:"
